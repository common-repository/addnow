== Description ==


== Installation ==

1. Upload `addnow` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to AddNow.com to register for an account. Follow the instructions on https://support.addnow.com/hc/en-us/articles/204075999-Getting-Started-with-AddNow to setup your site and add a Sharing Buttons Widget.
4. Go to Settings > Addnow to configure the Api key and the AddNow Widgets.
